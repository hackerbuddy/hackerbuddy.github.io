Acme SDK - Getting Started

Welcome to the Acme SDK documentation. This guide will help you integrate our payment processing API into your application.

Installation

npm install @acme/sdk

Quick Start

First, initialize the client with your API key:

import { AcmeClient } from '@acme/sdk';

const client = new AcmeClient({
  apiKey: process.env.ACME_API_KEY,
  environment: 'sandbox'
});

Making Your First Payment

Use the createPayment method to process a transaction:

const payment = await client.createPayment({
  amount: 1999,
  currency: 'usd',
  description: 'Order #1234'
});

For more details, see the full API reference.
